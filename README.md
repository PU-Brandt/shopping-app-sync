# PU-Brandt Home Assistant Add-ons

Dieses Repository enthaelt Home-Assistant-Add-ons fuer externe Python-Dienste.

## Add-ons

- ShoppingSync Control: Konfiguration, Status und Diagnose fuer den extern laufenden ShoppingSync-Dienst.

## Grundprinzip

Die fachliche Logik laeuft im externen Python-Dienst. Das Home-Assistant-Add-on stellt die installierbare Bedien-, Konfigurations- und Diagnoseoberflaeche in Home Assistant bereit.
